Rebel CMS

A fork of the discontinued Umbraco 5

Why?

We believe that "Umbraco 5" can be rescued. Lots of good code and functionality can be salvaged. We have to skills to fix it, and we will give it our best shot!

What we want to accomplish...

  • a brand new kick-ass responsive admin UI
  • REST API
  • Mono support
  • Heroku and Appharbor support
  • Performance issues sorted out
  • RavenDB and NoSQL core replacement
  • Continued development and support for "Umbraco 5" supporters

Who do you think you are?

We brought you: BBC iPlayer, BBC.com, BBC.co.uk, Topgear.com, Radiotimes.com, Mercuryrecords.com, Goodfood.com, Fictionrecords.co.uk, Gardenersworld.com

Yes, we read all the blog posts, we heard all the critical opinions, but call us crazy. We have rescued a LOT worse!

Open Source

Nothing will ever change that. This will be as Open Source as "Umbraco" is with the same licence. Contributors welcome!
